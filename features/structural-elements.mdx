## Structural Profiles

When leg-type supports include bracing members, PSI supports the following standard structural profiles:

| Profile Type | Designation Example | Description |
|---|---|---|
| **Angle (L)** | L2x2x1/4 | Equal-leg or unequal-leg angles used as diagonal braces |
| **Beam (W/I)** | W8x31 | Wide-flange or I-beam sections for heavier brace frames |
| **Pipe (Sch 40/80)** | Pipe 3" Sch 40 | Circular hollow sections with schedule-defined wall thickness |
| **Square Tube (HSS)** | HSS 4x4x1/4 | Hollow structural sections for compact braced frames |

Profile selection affects member capacity checks for axial load, bending, and combined loading per the applicable code.

## Braced Leg Configurations

For leg-supported vessels, brace members connect between the legs to resist lateral loads from wind and seismic events. The bracing layout — X-brace, K-brace, or diagonal — is configured in the Support Design page when the **Legs** support type is selected.

The brace profile designation (e.g., `L2x2x1/4` for a 2 × 2 × 1/4 inch angle) sets the section properties used in the capacity calculations.
