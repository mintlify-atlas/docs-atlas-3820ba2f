The Support Analysis page (`/soporte-analisis`) provides the results and analytics dashboard for pressure vessel support design. The page uses a responsive grid layout built on a set of analytics dashboard components drawn from the application's UI library.

<Note>
  The analytics dashboard is the results view for support design work. Navigate to the support-specific modules under **Elementos a presión** in the sidebar to configure design parameters before reviewing results here.
</Note>

## Dashboard Layout

The page renders a 12-column responsive grid with the following widget arrangement:

| Region | Component | Width |
|---|---|---|
| Top | `AnalyticsMetrics` | Full width (12 cols) |
| Second row | `AnalyticsBarChart` | Full width (12 cols) |
| Middle left | `TopChannel` + `TopPages` | 7 cols (xl) |
| Middle right | `ActiveUsersChart` | 5 cols (xl) |
| Lower left | `AcquisitionChannelChart` | 7 cols (xl) |
| Lower right | `SessionChart` | 5 cols (xl) |
| Bottom left | `DemographicCard` | 5 cols (xl) |
| Bottom right | `RecentOrderAnalytics` | 7 cols (xl) |

### Metrics Row

At the top of the page, the `AnalyticsMetrics` component displays a row of key performance indicator cards — summary figures derived from the current design state.

### Bar Chart

The `AnalyticsBarChart` component renders a full-width bar chart below the metrics row, comparing values across multiple categories.

### Channel and Pages Breakdown

Two side-by-side widgets in the center section:
- `TopChannel` — breakdown of contributions or categories by relative weight
- `TopPages` — list of most-active calculation routes or sessions

### Active Users Chart

The `ActiveUsersChart` widget on the right side of the center row presents time-series activity data.

### Acquisition and Session Charts

The lower section contains two additional visualizations:
- `AcquisitionChannelChart` — distribution of activity across input channels
- `SessionChart` — progression of results across calculation runs

### Demographic Card

The `DemographicCard` widget provides a geographic or categorical breakdown relevant to the current analysis context.

### Recent Activity Table

At the bottom, the `RecentOrderAnalytics` table lists recent design events in a tabular format with timestamps and status indicators, providing an audit trail of parameter changes.

## Workflow Integration

The Support Analysis page is the results view in the pressure vessel support workflow:

1. Configure vessel geometry, loads, and support parameters — available via **Elementos a presión → Diseño de soportes** in the sidebar
2. Review computed results and analytics here on the Support Analysis page (`/soporte-analisis`)
3. Export the formal calculation report using the print button on the design page

<Tip>
  The [Support Design](/features/support-design) page documents all available input components (GeometryCard, WindCard, SeismicCard, and the five support type panels) that feed results into this analysis view.
</Tip>
